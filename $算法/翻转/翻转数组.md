## 方法解析
### 遍历法
1. 创建一个新数组 ans[]
2. 从尾部开始遍历原数组, 对于遍历的元素, 依序放入 ans 中
## 代码实现
```JavaScript
function reverse(arr)
{
    if(!arr)
    {//若arr为空数组, 直接返回
        return arr;
    }

    const ans=[];
    for(let i = arr.length-1; i>=0; i--)
    {
        ans.push(arr[i]);
    }
    return ans;
}
```

### 分治法
1. 将数组均分成左右两部分，对每部分递归执行翻转操作.
2. 然后将左半部分拼在右半部分的后面，即完成了翻转。
