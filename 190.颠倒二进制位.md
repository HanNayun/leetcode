## 解答
### 逐位颠倒
1. ```n&1```, 获得 *n* 的二进制的最后一位
2. ```(n&1) << (31-i)```, 将获得的最后一位移到颠倒后的位置
3. ```ans = ans | ((n&1) << (31-i));```, 将其与 ans 进行或运算, 将其添加到ans中.
4. ```n = n>>>1```, 处理完最后一位后, 将前一位移到最后一位继续进行处理.
```JavaScript
/**
 * @param {number} n - a positive integer
 * @return {number} - a positive integer
 */
var reverseBits = function(n) {
    let ans = 0;
    for(let i = 0; (i<32) && (n>0); i++)
    {
        ans = ans | ((n&1) << (31-i));
        n = n>>>1;
    }
    //必须使用 ">>>" 用来转换格式
    return ans>>>0;
};
```
### 分治
>若要翻转一个二进制串，可以将其视为数组, 将其均分成左右两部分，对每部分递归执行翻转操作，然后将左半部分拼在右半部分的后面，即完成了翻转。   

**对于递归的最底层, 我们需要交换所有奇偶位:**
1. 取出所有奇数位和偶数位
2. 将奇数位移到偶数位上, 偶数位移到奇数位上.
3. 类似地，对于倒数第二层，每两位分一组，按组号取出所有奇数组和偶数组，然后将奇数组移到偶数组上，偶数组移到奇数组上。以此类推。
