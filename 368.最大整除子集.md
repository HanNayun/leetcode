给你一个由 **无重复** 正整数组成的集合 nums ，请你找出并返回其中最大的整除子集 answer ，子集中每一元素对 (answer[i], answer[j]) 都应当满足：   
- answer[i] % answer[j] == 0 ，或
- answer[j] % answer[i] == 0  
如果存在多个有效解子集，返回其中任何一个均可。

 
```
示例 1：
输入：nums = [1,2,3]
输出：[1,2]
解释：[1,3] 也会被视为正确答案。

示例 2：
输入：nums = [1,2,4,8]
输出：[1,2,4,8]
```


## 解析
本题使用动态规划, dp[i] 指 (nums[0]...nums[i]) 中的最最大整除子集的长度.  
已知, 如果 ```a%b == 0 && b%c == 0```, 那么必然有```a%c == 0```.  
根据该理论可以求dp[i]:
1. 首先将 nums 按升序排序
2. 由于每个数字都可以单独的作为一个最大整出数组, 因此初始化 dp[i] = 1 (i=0...n-1, n为nums的长度)
3. 双重循环求dp[], 
   ```JavaScript
    for i = 0 to n-1:
        for j = 0 to i:
            if(nums[i]%nums[j]==0):
                dp[i] = max(dp[j]+1, dp[i])
   ```
4. 由于数组nums是 **无重复** 数组, 因此最后的最大整除数组的对应的dp[i]组成的序列一定是 1, 2, 3....n-1 这样的序列. 
5. 从尾部遍历 dp, 首先找到 dp[i] 最大的位置i, 然后其对应的 nums[i] 应当是 nums 的最大整除数组中的最大值. 
6. 将当前的 dp[i] 值记为 maxSize, nums[i] 值记为maxValue.
7. 将 nums[i] 加入作为最大整除数组返回的 ans 数组中, ```maxSize--```. 然后继续遍历dp, 直到满足 ```dp[i]==maxSize && maxValue%nums==0)``` 若找到了满足条件的 i, ```goto``` (6). 否则遍历结束.
8. return ans.

```JavaScript
/**
 * @param {number[]} nums
 * @return {number[]}
 */
var largestDivisibleSubset = function(nums) {
    const n = nums.length;
    //注意, 由于大部分情况下 JavaScript 会默认数组为字符串数组, 因此需要手动写比较函数排序来得到正确的排序
    nums.sort(function(a, b){return a-b});
    var dp=new Array(n).fill(1);
    for(let i=0; i<n; i++)
    {
        for(let j=0; j<i; j++)
        {
            if(nums[i] % nums[j]==0)
            {
                dp[i] = (dp[j]+1 > dp[i]) ? dp[j]+1: dp[i];
            }
        } 
    }

    var ans = new Array();
    var maxSize = Math.max(...dp);
    var maxValue =0;

    for(let i=n-1; i>=0; i--)
    {
        if(dp[i]==maxSize && maxValue % nums[i]==0)
        {
            ans.unshift(nums[i]);
            maxSize--;
            maxValue=nums[i];
        }
    }

    return ans;
};
```
