# 461. 汉明距离
>两个整数之间的汉明距离指的是这两个数字对应二进制位不同的位置的数目。  
>给出两个整数 x 和 y，计算它们之间的汉明距离。

**注意**: $0 \le x, y <2^{31}$  
**示例**:
> **输入**: x=1, y=4  
> **输出**: 2

## 代码
```Java
class Solution {
    public int hammingDistance(int x, int y) {
        int xor = x^y;
        int count = 0 ;

        while(xor!=0)
        {
            count++;
            xor=xor&(xor-1);
        }

        return count;
    }

    public int hammingDistance(int x, int y) {
        int xor = x^y;
        int count = 0 ;

        while(xor!=0)
        {
            if(xor%2==1)
            {
                count++;
            }
            xor=xor>>1;
        }
        return count;
    }


}
```
