## 解析
动态规划 dp[i] = dp[i-1] + dp[i-2]  

[link](https://leetcode-cn.com/problems/climbing-stairs/)

```JavaScript
/**
 * @param {number} n
 * @return {number}
 */
const dp = new Array(46).fill(0)
dp[1] = 1
dp[2] = 2
for (let i = 3; i <= 45; i++) {
    dp[i] = dp[i - 1] + dp[i - 2]
}

var climbStairs = function (n) {
    return dp[n]
};
```
