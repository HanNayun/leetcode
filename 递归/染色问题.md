## 染色问题
>给一个$n*m$的方格，有 *c* 种颜色的染料，每一种染料有 *c[i]* L，每涂一个格子需要1L，对于某个格子上下左右4邻域的格子不能染成和本身格子一样的颜色，求是否能够找到染色方案？（保证升数总和和格子总数一致。）

>解析：  
格子坐标之和的奇偶性与四邻域不同，所以根据抽屉原理，如果 $max{C[i]> (n*m+1)/2}$ 一定是无解的

```Java
public class Main
{
    static boolean tag = false;

    static void canDyeing(int[][] map, int[] color, int x, int y)
    {
        if(y>=map[0].length)
        {
            y=0;
            x++;
        }

        if(x>=map.length)
        {
            tag=true;
            return;
        }

        for(int i = 0; i<color.length && !tag; i++)
        {
            if(y <= 0 || (color[i]>0 && map[x][y-1] != i))
            {
                if(x <=0 || color[i]>0 && map[x-1][y] != i)
                {
                    map[x][y] = i;
                    color[i]--;
                    canDyeing(map, color, x, y+1);
                    color[i]++;
                    map[x][y]=-1;
                }
            }
        }        
    }

    public static void main(String[] args)
    {
        int n=5, m=4;
        int[][] map = new int [n][m];
        for(int i=0; i<map.length; i++)
        {
            Array.fill(map[i], -1);
        }

        System.out.print(map);
        int[] color= {11,1,2,5,1};

        canDyeing(map, color, 0, 0);

        System.out.println(tag ? "Yes" : "No");
    }
}
```
